# 🟦 Polypropylene (T3)

> [!INFO] Tier 3 plastic
> Tougher and more heat-tolerant than polyethylene, polymerized from propylene.

---

## Main recipe - Propylene Polymerization

```mermaid
flowchart LR
    PRO[Propylene: 30] --> M{{Polymerizer}} --> PP[Polypropylene: 30]
```

|          | Input        | Output           | Building    | Time |
| -------- | ------------ | ---------------- | ----------- | ---- |
| **Main** | 40 Propylene | 20 Polypropylene | Polymerizer | 4 s  |

---

## Alternate 1 - Heavy-Cut Cracking

Pull propylene from a heavier cut for a higher batch yield.

| Input                       | Output           | Building    | Time |
| --------------------------- | ---------------- | ----------- | ---- |
| 40 Refinery Gas + 10 Diesel | 10 Polypropylene | Polymerizer | 8 s  |

---

## Alternate 2 - Copolymer Blend

Blend with polyethylene for a balanced impact-resistant copolymer.

| Input                          | Output           | Building | Time |
| ------------------------------ | ---------------- | -------- | ---- |
| 30 Propylene + 15 Polyethylene | 45 Polypropylene | Blender  | 6 s  |
