# 🟣 Crude Plastic (T1)

> [!INFO] Tier 1 plastic
> Refinery gas force-polymerized into a rough, brittle plastic. 
> Good enough for crates and casings

---

## Main recipe - Crude Polymerization

```mermaid
flowchart LR
    RG[Refinery Gas: 40] --> M{{Refinery}} --> CP[Crude Plastic: 20]
```

|          | Input           | Output     | Building | Time |
| -------- | --------------- | ---------- | -------- | ---- |
| **Main** | 40 Refinery Gas | 20 Plastic | Refinery | 6 s  |

---

## Alternate 1 - Naphtha Set

Polymerize light naphtha instead of gas - slightly better yield.

| Input      | Output     | Building | Time |
| ---------- | ---------- | -------- | ---- |
| 30 Naphtha | 20 Plastic | Refinery | 6 s  |

---

## Alternate 2 - Resin Casting

Cure aromatics into a hard thermoset resin (think bakelite).

| Input                   | Output     | Building  | Time |
| ----------------------- | ---------- | --------- | ---- |
| 20 Aromatics + 10 Water | 25 Plastic | Assembler | 8 s  |
